Hello World!

My Little Blog is my personal site project! It is a site written in Python using the Flask framework and runs inside a Docker container. Nginx is configured as a reverse proxy and serves inbound traffic to the container. The details of this setup are/will be highlighted here in the various pages of the blog posts section.

Everything on this site is a work in progress! Part of the fun is getting something started and building it along the way.

Some basic early features I want to start on are pages and posts. Pages will be basic website staples like about, contact, etc while Posts would be your traditional blog posts. I'd like to have the posts be written in mardown and ingested into the site for render.
